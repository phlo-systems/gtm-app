# GTM — Global Trade Management

**The trade intelligence platform for physical commodity trading.**

Built by [Phlo Systems](https://phlo.io) | London, UK

## What is GTM?

GTM enables anyone in the physical commodity supply chain to model trade costs, validate incoterm obligations, calculate customs duty, and track deal profitability — either standalone or integrated with their ERP.

## Features

- **Pre-Trade Feasibility** — Cost matrix, incoterm gap analysis, margin governance
- **Customs Intelligence** — HS code lookup, duty & VAT across 17 jurisdictions
- **Post-Trade Analytics** — Estimated vs actual variance, deal-level P&L
- **ERP Integration** — Xero, Sage, Acumatica (draft PO push on deal approval)
- **Standalone Mode** — Works without any ERP connection

## Tech Stack

- **Framework**: Next.js 14 (React)
- **Hosting**: Vercel
- **Auth**: Coming soon (Supabase)
- **Payments**: Coming soon (Stripe)
